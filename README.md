# ♻ EcoSort AI

> Turning waste into responsibility using AI 🌱

## 📌 Overview

EcoSort AI is a smart waste segregation system that helps users identify different types of waste and guides them for proper disposal.
The goal is to promote sustainable habits and reduce environmental pollution.

---

## 🚀 Features

* 🔍 Waste Detection (AI-based concept)
* 🗑 Disposal Guidance (Which bin to use)
* 📊 Eco Score Tracking
* 🎨 Modern Dark UI with Card Navigation
* 🌐 Multi-page Website (Clean UX)

---

## 🛠 Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Python (Flask)

---

## ⚙ How It Works

1. User uploads an image
2. System detects waste type (simulated AI)
3. Displays waste category and disposal instructions
4. Eco score increases to encourage sustainable behavior
